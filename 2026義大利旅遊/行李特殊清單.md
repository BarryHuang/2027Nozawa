# 🎒 多洛米蒂特殊行李清單 (Dolomites Special Packing List)
*針對本次「親子高山健行＋租車自駕」行程的特製清單，已排除基本衣物與盥洗用品。*

## 🥾 健行與高山裝備 (Hiking Gear)
- [ ] **防滑健行鞋 (必須)**：強烈不建議穿一般平底休閒鞋（如小白鞋）。Seceda 的一路下坡和巨石城的石頭路段需要抓地力好的鞋子（越野跑鞋或輕量登山鞋即可），避免長輩或小孩滑倒。
- [ ] **登山杖 (Trekking Poles)**：至少備妥大人用的 1-2 雙。Seceda 連續下坡路段對膝蓋負擔極大，有登山杖支撐會輕鬆很多；若遇下雨路滑更是保命神器。
- [ ] **防風防水薄外套 (Gore-Tex 或同級)**：7 月的多洛米蒂雖然平地熱，但山上 (2500m+) 溫差大，且棺材纜車是四面通風的，在上面會冷。採「洋蔥式穿法」。
- [ ] **好背的小雙肩包 (Daypack)**：每天出門只需要裝水、零食和防曬乳，雙手要騰空出來牽小孩或拿登山杖。

## ☀️ 防曬與高海拔對策 (Sun Protection)
*高海拔紫外線非常毒，且周圍沒有樹蔭遮蔽（尤其 Seceda 與休斯高原）。*
- [ ] **高係數防曬乳**：需隨身帶小罐的在山上補擦。
- [ ] **防紫外線墨鏡 (全家都要)**：保護眼睛，特別是在有大面積自色岩壁反光的地方（如蓋斯勒群峰）。
- [ ] **護唇膏 (具防曬係數尤佳)**：高山空氣非常乾燥，容易脫皮甚至裂開。
- [ ] **寬簷遮陽帽 / 附帶綁帶的帽子**：山上風有時很大，帽子如果沒有綁帶很容易飛走。

## 🚗 自駕與文件裝備 (Car & Logistics)
- [ ] **國際駕照 + 台灣駕照正本**：租車必備（義大利常會要求看正本）。
- [ ] **預訂時的信用卡 (VISA 108 結尾這張)**：**【極重要】** 租車公司現場會要求用原本這張卡刷取 500 歐元的保證金 (押金)，若沒帶可能無法順利取車。
- [ ] **手機車用磁吸架 / 支架**：山路髮夾彎多，能穩固看導航非常重要！
- [ ] **車用「傳輸線」(必備數據傳輸功能)**：因為您的租車有附 GPS 和 **Apple CarPlay / Android Auto**，強烈建議帶原廠或可以傳輸資料的線材，直接把手機導航投射到車機螢幕上最安全。
- [ ] **實體印出的確認信合集**：包含 Zanser Alm 的停車場通行證、Seceda 纜車票、Pitzock 晚餐訂位信等。山裡網路訊號有時不佳，實體紙本最保險。
- [ ] **零錢硬幣包 (超重要)**：很多高山山屋的廁所或私人停車場（如 Rifugio Valentini 的 10 歐元）**只收現金硬幣**。
- [x] *(不用帶)* 兒童增高墊：您已經在 SIXT 訂單中加購（54.62 歐元），租車公司會準備，不需要從台灣扛去。

## 👧 小孩與家庭特殊物資 (Kids & Family)
- [ ] **泳衣 / 泳具 (必備)**：Ortisei 鎮上的飯店通常有附設溫水泳池，或者入住時會贈送鎮上 **Mar Dolomit 公共水上樂園** 的免費入場券。這絕對是每天傍晚提早下山後，讓小孩放電、大人泡湯最棒的活動！
- [ ] **沿途哄騙用零食/補給品**：國外山屋賣的零食不一定符合胃口。建議從台灣帶一些小孩最愛的軟糖、小餅乾，在步道(如 Trail 35 或巨石城) 走不動時當作「里程碑獎勵」。
- [ ] **輕巧保溫瓶**：就算夏天，能讓長輩如果在山頂喝到一口熱水也是很棒的。
- [ ] **輕便雨衣 / Poncho**：不佔空間。如果突遇高山午後雷陣雨，可以马上套上保護小孩避免失溫。
- [ ] **防蚊液 / 蚊蟲咬傷藥膏**：夏季山林的草甸區域（如富內斯山谷和安娜谷溪邊）可能會有蚊蟲或蒼蠅（很多牛大便）。
- [ ] **簡易急救包**：OK繃（最重要）、優碘棉片、小護士，用來對付不小心在巨石城擦傷或跌倒。
